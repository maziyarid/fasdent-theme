# Fasdent – Featured Image & Supporting Image Generation Prompts
**Version:** 2.2.1  
**Brand colours (must respect):**  
- Primary cyan: `#09D4D6`  
- Primary blue: `#0E55B1`  
- Deep navy: `#071F3F`  
- Background / soft surfaces: `#F7FAFC`, pure white, soft pale-blue  

**Visual style rules (apply to every prompt):**  
- Calm, premium, modern dental/healthcare aesthetic  
- Clean, bright, professional, trustworthy  
- Soft natural lighting, subtle depth of field  
- No blood, no graphic treatment, no pain, no fear expressions  
- No malformed anatomy, no distorted hands/teeth  
- No text, no watermarks, no logos (except when explicitly instructed to leave space for Fasdent logo in bottom-right)  
- RTL-friendly composition (important subjects slightly centre-left or centred)  
- Landscape 16:9 preferred for featured images (1600×900 target, safe crop to 1200×675)  
- Keep key content inside central 80 % safe area  

**Usage:** Replace only `[TITLE]` (or the specific subject). Keep the rest of the prompt intact.

## Delivery Notes

1. Generate images at **1600×900** (or higher) in 16:9.
2. After generation, convert to WebP for performance.
3. Upload to Media Library → set as Featured Image.
4. For the real clinic photos already prepared (`assets/images/clinic/`), use them directly for Gallery and About sections.
5. Always keep the central 80 % safe for cropping on mobile.

These prompts are optimised for Midjourney, Flux, DALL·E 3, Ideogram, and similar tools.
